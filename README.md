# Cdecl-telugu

Cdecl-telugu is a tool for translating c declarations to telugu.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirments.

```bash
pip install -r requirements.txt
```

## Usage

```bash
make
```
```bash
python -u translator.py
```
